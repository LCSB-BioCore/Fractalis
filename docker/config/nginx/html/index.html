The fact that you see this message means that Fractalis has been successfully deployed via the provided Docker image. This system serves from now on as a computational endpoint for the Fractalis stack.


About

Fractalis is a service and a library that work in tandem to equip existing platforms with visual analytical capabilities for explorative data analysis. This is achieved by isolating all platform-dependent code into a single layer, which is responsible for migrating currently needed data into the Fractalis analysis cache.
